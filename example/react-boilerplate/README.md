# react-boilerplate

## Scripts

* `npm start`:
  install dependencies and build the development version
* `npm run preview`:
  build the production version and use browser-sync to preview
* `npm run build`:
  build the production version

## Dependencies

* `jquery`
* `lodash`
* `moment`
* `react`
* `react-dom`
* `react-router`
* `reflux`
